# WLIC6001 Project

This repository contains materials for the **WLIC6001** course project.

## Project Overview

The main deliverable is a research proposal exploring topics in **higher education**, with a focus on **quantitative data sources**.  
The proposal is written in HTML format for easy viewing and preservation of formatting (tables, headings, references, etc.).

Key elements include:
- Clear research objectives and questions
- Discussion of relevant quantitative data sources in higher education
- Proposed methodology (likely survey-based, secondary data analysis, or statistical approaches)
- Literature context and justification

This work was developed as part of coursework or an independent research component.
